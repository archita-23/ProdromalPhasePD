Prodromal Phase Detection for

Parkinson's Disease

"A data-driven system that detects early abnormal health patterns during the prodromal phase of neurological diseases, especially Parkinson’s disease, using longitudinal behavioral data and unsupervised anomaly detection."

01. Problem Statement

Neurological diseases like Parkinson’s are usually diagnosed only after visible motor symptoms appear, by which time significant neuron damage has already occurred. Early prodromal symptoms such as sleep changes, mood variation, and fatigue are subtle and rarely analyzed together. Click to view Literature Review.

02. METHODOLOGY

Technical Approach

The system collects longitudinal behavioral data such as sleep duration, mood variation, activity level, and fatigue. After preprocessing and normalization, relevant features are extracted to represent daily behavioral patterns. An unsupervised anomaly detection model (Isolation Forest) learns the user’s normal baseline behavior and continuously evaluates new data. When multiple parameters deviate simultaneously beyond normal variation, the system flags a potential risk window. The output is presented through visual trends and anomaly indicators for early awareness.


Methodology Steps:

-Data collection (simulated or recorded daily behavioral parameters)

- Data preprocessing and normalization

- Feature selection and time-series trend analysis

- Training unsupervised anomaly detection model

- Detecting multi-parameter deviations

- Generating risk flags (non-diagnostic)

- Visualizing behavioral patterns and anomalies

03. TECHNICAL STACK

System Architecture

The system is built using Python and JavaScript. The backend uses scikit-learn for anomaly detection and pandas for data processing. The frontend is developed with HTML, CSS, and JavaScript to visualize behavioral trends and anomalies. Data is stored in a local database for persistence across sessions.


Key Technologies:

- Python

- Data Processing: Pandas, NumPy

- Machine Learning: Scikit-learn (Random Forest, SVM)

- Time-Series Analysis: Statistical trend detection

- Visualization: Matplotlib, Plotly, Streamlit

04. RESULTS

Expected Outcomes

The system successfully learned normal behavioral patterns and detected periods where multiple parameters showed abnormal deviation. The anomaly detection model identified a continuous risk window corresponding to gradual behavioral drift rather than isolated fluctuations. Visual outputs highlighted these deviations clearly, demonstrating that longitudinal pattern monitoring can reveal early abnormal trends. The results confirm that combining multiple subtle changes is more effective than observing single parameters for early risk awareness.


Key Results:

- Model established a personal normal baseline

- Detected sustained abnormal behavior period

- Reduced false alerts from single-parameter changes

- Identified multi-signal deviation patterns

- Visual graphs clearly showed risk windows

- Demonstrated feasibility of early non-diagnostic monitoring

05. ETHICAL CONSIDERATIONS

Privacy and Data Handling

System does not provide medical diagnosis.


- Results are for educational and research purposes only.

- Uses anonymized or simulated data.

- Avoids alarming predictions and ensures responsible reporting.

- Non-diagnostic output presentation
